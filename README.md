java application

Creates a UF2 File with some parameters
e.g. WLAN SSID PSK
which can be loaded by the Pico Bootloader
into the top of the Raspberry Pi Pico QSPI Flash
on a fixed address.

Here: 0X101FF000 to 0X10200000 (last 4096 Byte Block)
